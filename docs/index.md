# SMTH Help Hub

Welcome to the Help Hub site for SMTH. Browse the guides below or add a new Markdown page to expand the help center.

## Documentation pages

- [Getting Started](index.html?file=getting-started.md)
- [Usage](index.html?file=usage.md)
- [FAQ](index.html?file=faq.md)

## Add a new page

1. Create a new Markdown file in the docs folder.
2. No need to update any navigation it will be automatically added to the page list.

You can also add code blocks and lists directly in Markdown.

Run locally using this terminal command

```bash
python3 -m http.server --directory docs 8000
```
